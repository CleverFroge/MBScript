WordEncode.py，用于生成字的码表：
	运行环境：Python 3
	Excel码表文件名：encode.xls
	码表格式：
		A列：繁体简体（f,j）
		B列：汉字
		C列：等级（t1~t5，或者空）
		D~J列：编码（标准、容错、简码）
		读取的总行数可以在代码第34行修改（0~30791，再后面的字加到里面输入法读不出来会出问题)
	完成后，会在当前目录下生成四个文本文件，手动将其改成Unicode，之后才能被输入法程序使用


WordEncode.py，用于生成字的码表：
	运行环境：Python 3
	形码码表文件名：phrase.xls
	码表格式：
		A列：汉字
		C列：第一形码
		D列：第二形码
		读取的总行数可以在代码第31行修改（0~31014）
	词组文件：phrase.txt
	词组文件格式：
		每行一个词组
	完成后，会在当前目录下生成Shape.txt，手动将其改成Unicode，之后才能被输入法程序使用
